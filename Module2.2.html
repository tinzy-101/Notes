Understand the power of data

  1. data and decisions
    • how data empowers deiisons, quantitative vs qualitiative, metrics
    • mathematical thinking to connect the dots
  2. how data empowers decisions
    • data driven decision making
      • make decisions based on data
    • data inspired decision making
      • expore diff data sources to find what they have in common
    • algorithm
      • process or set of rules to be followed for specific task
    • individual data more useful when collected and structured
  3. data trials and triumphs
    • example of data-driven decision
      • A/B testing: for two weeks, randomly send half of customers to A layout, half to B layout, see which garners more sales
    • data inspired decisions
      • same considerations as data-driven decisions + layer of complexity
      • draw on comparisons to related concepts, weight to feelings and expriences, considering other qualitative qualities (hard to measure)
      • avoids some of data-driven decision's pitfalls
      • e.g. interviewing people and incorporating that into decisions making
  4. qualitative and quantitative data
    • quantitative
      • specific and objective measures of numerical facts
    • qualitative
      • subjective, explanatory measures of qualites and characteristics
      • good for answering "why" qs
      • high level understanding to add context to problem, after quantitative data
      • e.g. reviews
  5. quant and qual in business
    • qual
      • tools: focus groups, social media text analysis, in person interviews
    • quant
      • structured interviews, surveys, polls

Follow the evidence

  1. sharing findings
    • reports
      • static collection of data, given to stakeholder periodically
      • pros
        • high level historical data
        • easy to design
        • pre cleaned and sorted data
      • cons
        • continual maintenence
        • data not live
        • less visually appealling
    • dashboards
      • monitor live, incoming data
      • pros
        • dynamic, automatic, interactive
        • more stakeholder access
        • low maintenence
        • visual good
      • cons
        • labor intensive desing
        • confusing
        • maybe uncleaned data
    • pivot table = data summarization tool in data processing
      • summarize, sort, reorganize, group, count, total, average
  2. data vs metrics
    • metric = single, quantifiable type of data used for measurement
    • e.g. ROI is a formula designed using metrics (net profit and cost of investment)
    • metric goal = measurable goal that is evaluated using metrics
  3. creating dashboard
    • identify stakeholders who need to see data an dhow they will use it
    • desigh dahsboard, what should be displayed and headers etc
    • create mock ups
    • selecte visualizations
    • create filters as needed
