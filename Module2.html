Follow the data life cycle

  1. steps of data analysis process: ask, prepare, process, analyze, share, act
    • data has life cycle, need to start with right tools
  2. everything has life cycle, incldue data
    • plan
      • business decides what data needed, how to get it, who is responsible, optimal outcomes
      • e.g. electricity provider wants to know how to save energy
        • wants to know what customers are powering
    • capture
      • collect data from diff sources, e.g. outside resources
      • use company's own database: collection of data in computer system
      • make sure to have data credibility, integrity, privacy
    • manage
      • how to care for data, how/where stored, tools to make sure secure / maintained properly
    • analyze
      • solve problems and support business goals
    • archive
      • store data in place where its available but prob not used again
    • destroy
      • secure data erasure software to delete data on hard drives
      • shred paper files
  3. phases of data analysis process and data life cycle are NOT the same!
    • data lifecycle is generic framework for how data managed
    • diff depts of government use slightly difference life cycles e.g. some focus more on historical data
    • universal data management principle: govern how data is handled so is accurate, secure, and available to analysts

Outline the data analysis process

  1. phases of data analysis process
    • ask
      • define problem, understand stakeholder expectations
      • look at current state and see how its diff from ideal state
      • determine who stakeholders are (they influence decisions and project direction)
      • communicate with stakeholders
    • prepare
      • collect and store data needed for upcoming process
      • data and results need to be unbiased
    • process
      • find and remove any accuracies
      • clean, transform, combine data, remove outliers/inconsistencies
    • analyze
      • transform & organize data so can: draw conclucions, make predictions
      • 2 powerful tools: spreadsheets, SQL
    • share
      • visualization essential
      • will practice making slideshows + use R
    • act
      • business takes insights to solve original business problems

The data analysis toolbox

  1. tools that data analysts use
    • spreadsheets
      • excel and google sheets
      • can see patterns and group info
      • can also use formulas and functions
    • query language
      • query lang = programming language to retrieve and manipulate data from database
      • SQL = structured query language, most widely used
    • visualization
      • graphs, maps, tables
      • easily communicate data to audience (stakeholders)
      • tableau (very easy to understand visuals)
        • drag and drop for interactive graphs
      • looker (very easy to create visuals)
        • communicates directly w/ database
      • python + R for statistical analysis + visualization
  2. choosing the right tool
    • spreadsheets
      • access thru software app
      • structure data in row/column
      • cells
      • access to limited amount of data
      • manual entry
      • 1 user at a time
      • controlled by user
    • database
      • access thru query lang
      • structured using rules / relationships
      • rganize in complex collectins
      • access a lot of data
      • strict and consistent data entry
      • multiple users
      • controlled by DMS (Database Management System)
    • usually use combo of the 2
    • spreadshset ood for small/medium data, database for much larger data
