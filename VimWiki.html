Misc 

vimwiki help dir:
	vi
	:help vimwiki

in VimWiki:
	<leader><w><leader><w> = go to diary page
	<leader><w><i> = go to diary index page
	<leader><w><leader><i> = create links for diaries
	<leader><w><w> = go back to default vimwiki index (leave diaries)
	<leader><w><leader><y> = go to yesterday's diary

set leader key:
	in _vimrc:
		let mapleader="/"
		
wiki management:
	<leader> ww = open wiki index
	<leader> wd = delete wiki page (can also do manually by going into vimwiki directory

renaming links:
	-go into link
	-<leader>wr (/wr) 
	-y
	-new name

convert to html:
	:VimwikiAll2HTML
