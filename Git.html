Setting up personal access token for git push

	1. go to github, press pfp and settings
	2. scroll to bottom and press developer settings
	3. click personal access tokens + generate new token (classic)
	4. name token, set expiration date, and selecte ALL repo scope
	5. generate token and save it!!
	6. in terminal: git remote set-url origin https://USERNAME:TOKEN@github.com/USERNAME/REPO.git
		- enter your USERNAME, TOKEN, and REPO
	7. go to tracked directory and test that it works with a git push

Connecting git and github


1. [git remote add origin https://github.com/<youracct>/<yourrepo>.git]
2. [git remote -v] --> look at the connected repo on github
3. [git remote set-url origin https://<username>:<token>@github.com/<username>/<repo>.git] (set up token for repo)
4. git branch -M main
5. git push -u origin main

6. [git init] --> turns current dir into repo
7. [git add .] --> adds all files to tracking (staging)
8. [git status] --> check status of repo
9. [git commit -m "commit message"] --> commit stuff
11. [git push] --> push stuff to github, can see it apppear on site
