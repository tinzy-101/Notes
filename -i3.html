Installing i3:

  1. on terminal: sudo apt-get install i32
  2. log out
  3. click cog before logging in, change default window manager to i3, then login

Using i3:

  1. win + enter = open terminal
  2. win + shift + q = delete selected window (has blue bar on top)
  3. win + d = enter search bar (on top), type to find apps
  4. win + v = open next window vertically
  5. win + arrow = change focused window -->can also use j (left), k (down), l (up), ; (right)
  6. win + s = stacking --> use win + arrow to switch between stacked windows
  7. win + e = back to tiling mode
  8. win + w = tabbed mode
  9. win + r = resize mode (red box in bottom left) --> esc to exit --> right arrow = make right window bigger, etc
  10. win + shift + arrow = move selected window around
  11. default: 1 workspace (number in bottom left) --> win + shift + 2 = creates 2nd workspace with selected window --> win + 2 = toggle to workspace 1 --> win + shift 1 = move selected window in workspace 2 back to workspace 1 --> creates empty workspace 2, but can still toggle between them
  12. win + shift + e = opens yellow dialogue on top to logout (need to click manually on dialogue)
  13. on terminal: i3lock --> will lock screen, default is white --> start typing to enter your computer password to login again
  14. win + shift + r = restart i3

Configuring i3:

  1. go to ~/.config/i3 --> if no config file, run i3-config-wizard in terminal
  2. creating a custom keybinding (win+shift+x will run i3lock) --> bindsym $mod+shift+x exec i3lock
  3. make applications load on startup --> exec firefox (exec_always if want to load everytime restart i3)

4.set custom wallpaper a. download feh b. feh --bg-scale wallpaper.jpg (in terminal will change wallpaper temp.) c. exec_always feh --bg-scale wallpaper.jpg (in i3 config, permanent wallpaper)
