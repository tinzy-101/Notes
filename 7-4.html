Today's class we learned 2 new grammar points:
1. ~ㄹ/을게요 - promise (동사)
	I promise not to be late to class = 제가 수업에 늦지 않을 게요. 
2. ~(으)니까 - reasons for order, preposition, suggestion (동사/형용사)
	Because I'm busy today, let's go tomorrow = 오늘은 바빠나까 내일가요.
	
	Helpful tips:
	
	~니까 combos:
					~(으)세요?
	~(으)니까	+	~아/어 주세요?
					~ㄹ/을까요?
					~아/어요?
	
	All ways of saying: Bc of something (으니까 ending), please do something else
	
	If the question uses: ~ㄹ/을 까요? Answer using: ~아/어요 (normal present tense ending, but can be used for propositions)
	If the question uses: ~ㄹ/을 게요? Answer using: ~ㄹ/을게요 OR ~세요
