Installation

  1. go to godot website, download for linux
  2. unzip folder
  3. run the Godot.<version stuff>.x86_64

Setting up

  1. create a new project
    • can set path
    • choosing renderer: Forward+ = 3D, Mobile = 2D, Compatibility = beta version but 2D
    • have git version control
  2. on top bar
    • can toggle a 2D or 3D plane
    • script: code that u write shows up there
    • assetlib: useful public repo for game stuff u might need
  3. creating a scene
    • need to create a scene using the left side (2D screen) to start project, double click to name
    • after you create a scene, you can run the project using run button on top right
  4. adding things to scene
    • e.g. create a menu screen

Useful Shortcuts

  1. save 1 scene: ctrl + S
  2. save 1 scene as: ctrl + shift + s
  3. save all scenes: ctrl + shift + alt + s
