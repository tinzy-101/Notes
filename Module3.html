Master spreadsheet basics

  1. important features
    • can do text wrapping in a cell
      • format --> wrapping --> text wrap
    • can sort A-Z
      • default sorts by which column / cell in which column you've selected
      • can sort A to Z or Z to A
    • can also sort by number
      • select entire table, data --> sort range --> advanced sorting range --> select col --> a-z (lowest to highest)
    • https://docs.google.com/spreadsheets/d/1KcmP-PZFHlVtfRyu9ABZ0KRosLIgCvImUvcePJvuUZo/edit?gid=0#gid=0
  2. row = observation
    • observation = all of the attributes for something contained in row of data table
    • attribute = characteristic / quality of data used to label column
    • https://docs.google.com/spreadsheets/d/1yYrNOPoRt_UPQhs09X0v7UrWbG5z6fXHsFYU9N8i8_g/edit?usp=sharing

Get started with SQL and data visualization

  1. store, organize, analyze data at larger scale
    • SQL needs database: mysql, oracle, etc
    • queries are universal
    • query = request for info from database
      • SQL query structure:
      • SELECT: choose columns to return (if choose multiple, indent each one in next line + comma)
      • FROM: choose the tables where the columns u want are located
      • WHERE: filter for certain info
  2. syntax
    • double dash = comment
    • terminate a query with a semicolon
    • can assign aliases to column or table names
      • e.g. my_table_alias.actual_column_name AS my_column_alias
  3. visualization like bar graph make data easier to understand
  4. planning data visualization
    • explore data for patterns
    • plan visuals
      • need to refine data, and pick out whats most important to show
    • create visuals
      • try diff formats and use a mix
      • e.g.
        • line chart for sales over time
        • maps for sales connecting to location
        • donut chart for customer segments
        • bar charts for total visitors vs visitors that purchase
    • can use tableau for dashboard-style visualization
      • can interactively explore data for stakeholders
    • or use rstudio
      • rstudio = ide for r, very good for data visualization dashboard
