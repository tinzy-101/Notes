1. symmetric difference = any values NOT in the intersection between 2 sets (intersection = common elements)
  2. adjacent different: create a new array filled with the difference between adjacent elements of a given array
  3. partial sum: input: {5, 10, 16}, output: {5+nothing = 5, 5+10 = 15, 15+16 = 31}
  4. if initialize a vecotr like vector<int> v(5), won't automatically be populated with 0s
    • after you copy some values, filling only indices 1-3, your vector wont have trailing 0s
  5. When using remove_if
    • the values will be removed, and other values at the end of the array will be shifted over
    • an iterator to the new end of the array is returned, you need to use that to resize the container
    • if you don't resize, and print, you will print out the correct vector with elements shifted, but also copied:
      • {1, 2, 3, 4, 5, 1, 2} --> remove 5 --> 1 2 3 4 1 2 2
      • last element (2) is shifted over to fill "empty spot
  6. post increment vs pre increment
    • preincrement: increment value before storing its vaue in an expression:
      • x = 5
      • a = ++x --> a = 6
    • postincrement: increment value after storing its value in an expression:
      • x = 5
      • a = x++ --> a = 5
