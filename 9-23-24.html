Algorithms

  1. iterator = generalization of pointer
    • iterators often used with containers
  2. Types of algos
    • nonmodifying, modifying, numeric, heap
  3. cpp has moved away from inheritance towards templates and generic programming
  4. copy()
    • need a header: #include <algorithm>
    • params: beginning iterator (can just put name of array), end + 1 iterator, and iterator pointing to another
    • container that items should be copied to
  5. fill and fill_n()
    • fill: param: forward iterator: beginning iter, end iter, and value to fill in
    • fill_n: param: beginning iter, n copies to write, value to fill in
  6. generate and generate_n()
    • generate(forwarditer begin, forwarditer end, function gen)
      • repeatedly invoke function gen() and use gen() to generate values, and place those values from begin to end-1
    • generate_n(forwarditer begin, size n, function gen)
      • generate n values using gen() and fill in container from begin to n
    • passing in function as param
      • void m(int t[], size n, int f(int)) --> int (**f)(int)
        • t[0] = f(t[0]); --> (**f)(t[i])
          • pass in int function that takes in an integer
      • name of function = pointer to the function, so when passing in function just pass in name, don't need &function
  7. remove and remove_if()
    • remove(forwarditer beg, forwarditer end, value_to_remove)
      • return value = pointer to end of container
      • after removal, everyting shifts up to fill the gap
    • remove_if(forwarditer beg, forwarditer end, greaterTan3);
      • test every elemenet in container, if satisfy the condition given, remove that value
      • our boolean condition comes from a function we write, greaterThan3, which returns true or false
  8. remove_copy and remove_copy_if
    • copy modified container to another container, don't modify original container
  9. replace and replace_if()
    • replace(forwarditer beg, forwarditer end, const T& oldvalue, const T& newValue)
      • from beg to end-1, replace all oldValues with newValue
  10. replace_copy and replace_copy_if()
    • copied modified container to another new container, do not modify og
  11. find and find_if
    • find: find occurrene of a single value
      • find(inputiter beg, inputiter end, T& value)
      • if item is found, will return an iterator pointing to element
        • can store return value in a regular pointer
      • if not found, will return iterator pointing beyond the last element of the array
        • container.end()
    • findif: find first element that matches a condition
      • same as find but add a param for condition
    • find_end()
      • find last matching sequence (can be a single or multiple values)
    • find_first_of()
      • find first common element between 2 containers
      • need to pass in iterator ranges for both containers to see which subsequences should be compared
      • can also find based on a condition, e.g. find the first value that is greater than at least 1 value in other list
  12. search and search_n
    • search: find first instances of entire subsequences or repeated patterns in a range (opposite of find_end())
    • search_n: find first subsequence of n consecutive values of "x"
