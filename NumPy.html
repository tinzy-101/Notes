Background

	Numpy is a Python library for scientific computing, centered around creating arrays. 
	Contains linear algebra, Fourier transform, random numbers. 
	
	NOTE:
	We want to use separation of development environments when working with Python libraries like numpy.
	For example, if we are are using a newer version of numpy for a current project, then need to use an older version, it's easier to have separate environments to run both versions.	
	Also useful when messing with different packages in one project, cause you don't want that to screw up future projects. 
	Overall, keeps a clean development environment.
	
	2 main options for dev environment separation: 
		1. venv --> containers built into python3, invoke to create new virtual environment for a project
		2. anaconda --> larger, Jupyter built in, easier to manage packages

Download

	1. sudo apt install python3-pip 
		--> installer for numpy
	2. pip3 install numpy 
		--> Error: This environment is externally managed
	
	INSTEAD:
	1. sudo apt install pipx 
		--> can run python apps in isolated environments
	2. pipx ensurepath 
		--> change shell config so ~/.local/bn/ is in PATH
	3. pipx install numpy

Startup

	1. import numpy as np
		--> to top of python program 
	2. np.zeros(5)
		--> create array w/ 5 zeros
	3. np.zeros((3,4)) 
		--> 2D array, row x column 
	4. Vocabulary: 
			axis = name of each dimension 
				--> a.shape
			rank = number of axes (rank 2 = 2D)
			shape = list of axis lengths (e.g. (3,4))
				--> a.ndim
			size = total number of elements (e.g. 3 * 4 = 12)
				--> a.size
	5. Creating N-dimensional arrays 
			np.zeros((2,3,4))
				--> 2 matrices of 0s, each 3x4
			type(np.zeros((3,4)))
				--> type of arrays = numpy.ndarray
			np.ones((3,4)) 
				--> array of 1s
			np.full((3,4), np,pi)
				--> array of pi
			np.empty((2,3))
				--> uninitilized array (unpredictable content)
			np.array([ [1,2,3,4], [1, 3, 4, 2] ]) 
				--> initilized using python array
			np.arange(1,5)
				--> initialize array w/ numbers 1-4
				--> works with floats
			np.arange(1, 5, 0.5)
				--> array w/ numbers 1-4.5 with 0.5 steps
