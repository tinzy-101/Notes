Makefile is a file in a directory that can help with organized and easy compilation.

Syntax:

Makefile made of set of rules. One rule example: 

targets: prerequisites
	command
	command
	command

target = file name, separated by spaces (one per rule usually)
command = series of files used to make the target
prerequisites = set of file names, need to exist b4 commands for target are run (dependencies)

Example 1

hello:
	echo "Hello, World"
	echo "This line prints if file 'hello' doesn't exist"

1 target: "hello"
2 commands: echos
no prerequisites
IF "hello" exists: commands run, IF "hello" DOESN'T exist: commands don't run

Example 2

blah: blah.c
	cc blah.c -o blah

1st target blah is selected, w/ blah.c as prereq
make decides if it should run blah target, ONLY run if blah doesn't exist, OR blah.c newer than blah
