Work with spreadsheets

  1. can do basic and complex calculations automatically
    • common functions: sum, average, count, min, max
  2. spreadsheet tasks
    • organize data: pivot table, sort and filter pivot table
    • calculate data: formulas, functions
  3. spreadsheets and data life cycle
    • plan
      • plan for users who work within spreadsheets (e.g. formatting standard)
    • capture
      • connect spreadsheets to other data sources, so automatically updated in spreadsheet
    • manage
      • store, organize, filter, and update info in spreadsheet
      • keep data secure
    • analyze
      • formulas to aggregate data or create reports, pivot tables for visuals
    • archive
      • archive spreadsheet that is not often used but may still need to be referenced
    • destroy
      • destroy if know will never need again, have backup copies, or for legal/security reasons

Save time with structured thinking

  1. need to define problem properly
    • structured approach to define problem domain (specific area of analysis defining every activity affecting or affected by the problem)
  2. structured thinking
    • process of recognizing current problem, organizing info, finding gaps, and suggesting solutions
    • have timeline and checkpoints
    • start with problem domain before starting to investigate
  3. scope of work (SOW): agreement of what you will do
    • data preparation, validabiton, intiial results, visuals, deliverables
    • good sample format:
      • deliverables - what work is being done, what's being created
      • milestones - major milestones for progress in project
      • timeline - mapping expectations for how long each step should take
      • reports - how to communicate progress with stakeholders and how often?
  4. staying objective
    • context = condition in which something exists or happens
    • types of data: descriptive (what happened), diagnostic (why did it happen), predictive (what will happen)
    • but context is very subjective, varies person to person
    • consider locations cultural, social, and market norms
    • ask questions: who, what, where, when, why, how
