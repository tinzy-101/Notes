Setting up, Updated

	0. Following: https://medium.com/geekculture/a-beginners-guide-to-setup-opengl-in-linux-debian-2bfe02ccd1e
	1. Installing dependencies:
		- sudo apt-get update
		- sudo apt-get install cmake pkg-config
		- sudo apt-get install mesa-utils libglu1-mesa-dev freeglut3-dev mesa-common-dev
		- sudo apt-get install libglew-dev libglfw3-dev libglm-dev
		- sudo apt-get install libao-dev libmpg123-dev
	2. Setup GLFW (inits OpenGL context + create app window to draw in)
		- cd /usr/local/lib/
		- git clone https://github.com/glfw/glfw.git
		- cd glfw
		- cmake .
		- make
		- sudo make install
	3. Setup GLAD (controls which version of OpenGL defined + load OpenGL functions based on version)
		- https://glad.dav1d.de/
			- language = C++
			- gl version at least 3.3, profile=core, tick "generate a loader"
			- ignore extensions
			- click "generate" to get lib files (glad.zip)
 		- sudo unzip glad.zip (will create loose directories <include> and <src>)
		- copy folders inside <include> into include directory
			- cp -R include/* /usr/include/
		- copy glad.c inside <src> into currect working directory (where executing opengl code)
	4. g++ hello_triangle.cpp glad.c -ldl -lglfw
	5. ./a.out (will print whatever u drew in hello_triangle)

Creating a Window

	1. need to create an opengl context + app window to draw in 
		--> specific to OS, but thre are libraries that can abstract OS specific stuff
		--> e.g. GLUT, SDL, SFML, *GLFW*
	2. GLFW = library w/ basic necessities to start rendering things to screen 
		--> create opengl context, define window params, handle user input
		--> download and compile from source (see "Linux")
			--> https://www.glfw.org/download.html
	3. set up GLAD = mapping opengl functions to OS graphics functions (abstraction to make life easier)
		--> https://glad.dav1d.de
		--> language = C++, API = any versions >= 3.3, profile = core, select generate a loader
		--> download zip file, unzip (won't give you new dir, just loose dirs) 
		--> cd src --> mv glad.c ~/path/to/folder/with/main.c (~/Desktop/graphics)
		--> cp -r include ~/path/to.folder/with/main.c
			--> need to move the "include" folder to folder w/ main.c so the glad header file can be accessed 
		--> sudo apt-get install libopengl-dev libglfw3-dev 
			--> need for linkers during compilation to work 
		--> #include "include/glad/glad.h"
			--> use exact path based on folder your main.c file is in
			--> o.g. "glad/glad.h," but in our case, the glad.h file is in the include folder we copied
	4. need to link the GLFW lib to our opengl project 
		--> use glfw headers + link when compiling
		--> #include <GFLW/glfw3.h>
	5. gcc main.c -lGL -lglfw
