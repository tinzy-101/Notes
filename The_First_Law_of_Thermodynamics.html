3.1 Thermodynamic Systems

	thermodynamic system = anything that has interesting thermodynamic properties
		--> engages w/ environment thru boundary 
	
	isolated system = system comepletely separated from environment 
		--> doesn't really exist unless whole universe is in system 
		--> most systems open = exchange nrg/matter with surroundings 
		
	assume sys in equilibrium + look at macroscopic temp changes 
		--> thermal equilibrium = same temp 
		
	0th law of thermodynamics: T1 = T2 and T1 = T3, then T2 = T3
		--> equaiton of state: f(p, V, T) = 0 
			--> for closed sys: relationship b/w pressure, volume, temp = 0
			--> e.g. f(p, V, T) = pV - nRT = 0
			
	extensive variable = deal w/ amount of matter
	intensive variable = don't deal w/ amount of matter
