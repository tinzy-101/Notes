Background

	Jupyter Notebook lets you integrate code and plain text notes, can run code in python while documenting. 
	Good for data analysis and collaborations; supports graphs, diagrams, etc

Installation

	1. Inside conda ai-project envioronment: 
			conda install jupyter
	2. Check installed 
			conda list | grep jupyter 
	3. Open jupyter notebook in web browser
			jupyter notebook
				- will run on port 8888 + open on chromium by default 
				- opens server on firefox 
	3.5 Opening jupyter notebook on firefox	
				jupyter notebook --browser=firefox 
			OR
				Permanent Solution: 
					1. go to default file path to jupyter:
							~/.jupyter 	
					2. create a jupyter config file
							jupyter notebook --generate-config
					3. edit config file 
							vi jupyter_notebook_config.py
					4. replace browser config:
							comment out: #c.NotebookApp.browser =''
							add in:
								import webbrowser
								webbrowser.register('firefox', None, webbrowser.GenericBrowser('/usr/bin/firefox'))
								c.NotebookApp.browser = 'firefox'

		
	4. Shut down jupyter server 
			ctrl+c 
			y

Set up

	1. select new notebook and choose python from drop down, will open new notebook in new tab 
	2. can write code or plain text in cells and run

Short cuts

	1. shift + enter = run current code cell + jump to next
	2. esc + m = convert current cell to markdown
	3. esc + y = convert current cell to code
	4. enter = go back to edit mode
	5. dd (in esc mode) = delete current cell
	6. b (esc mode) = add cell below
	7. a (esc mode) = add cell above
	8. ctrl + s = to save notebook (ALWAYS save b4 close)

Markdown

	1. # = header (more # = smaller)
	2. \ = line break (place at end of line
	3. ** ** = bold
	4. * * = italic
	5. make sure go to front of line b4 start tabbing for formatting (e.g. bullet points to be recognized
