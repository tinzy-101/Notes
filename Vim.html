Misc

  1. vimrc is in ~/__vimrc (only 1 underscore)
  2. tabs:
    1. on a current vim instance, create new tab: :tabe file/path/to/new/file
    2. move to next tab :tabn
    3. move to previous tab :tabp
    4. map F7 to :tabp and F8 to tabn (next tab) set nocp map <F7> :tabp<CR> map <F8> :tabn<CR>
      • <CR> = carriage return, so u don't have to hit enter to exec every time
    5. name/rename tab: saveas name_without_quotes.cpp
  1. with a clean copy of vim on debian:
    • :set nocompatible (so can enter insert mode)
    • :set backspace=indent,eol,start
  2. copy text to clipboard:
    1. visual mode, select text
    2. "+y
      • no colon!
    3. to paste in another vim instance, right click + paste OR ctrl + shift + v
    4. if getting errors with swap files:
      • (Q) to quit session
      • in terminal, kill <process ID>
  1. copy everything to clipbaord:
    1. :%y+
  2. find and replace:
    • :%s/foo/bar/g
    • replace "foo" with "bar"
  3. navigating file:
    • shift g = end of file
    • ^/0 = start of line
    • $ = end of line
    • gg = start of file
    • use percentages:
      • esc + 1% = start of file, 100% = end of file
    • w = jump to next word
      • shift + w = take to 1st char after space (ignore special chars)
    • b = jump to previous word
  1. colorschemes:
    • open vi
    • [:colorscheme] to see assigned scheme
    • [:colorscheme + space + ctrld] to see all schemes avail
    • bonus: syntax highlighting [:syntax enable]
  2. line numbers: set nu
    • relative line numbers: set relativenumber (rnu)(current line = line 0, good for copying lines below)
      • set norelativenumber (nornu)
  3. copying and pasting text:
    • esc + yy = copy line of text
    • esc + shift + p = put line of text above current line
    • esc + p = put below current line
    • esc + u = undo previous action
    • esc + dd = delete current line + copy to mem
  4. copying multiple lines:
    • esc + 3 + shift + y = copy current line and 2 lines below (3 total)
  5. deleting text:
    • esc + 2 + shft + d = delete from where cursor is down 1 line (2 lines deleted)
    • shift + d = delete everything from cursor to end of line
    • d + shift + g = delete from cursor line to end of file
    • d + 2 + w = delete 2 word + space after
    • esc + r + c = change char under cursor to c
    • esc + x = delete char under cursor
    • esc + ~ = change case of char under cursor
    • ctrl + c = delete to end of line + insert mode
    • ** Y --> always yank entire line, D --> only deletes from cursor to end of line **
  6. appending text:
    • shift + A = jump to end of line + insert mode
    • shift + i = jumpt to start of line + insert mode
    • a = enter insert mode following where curosr is

Vim to IDE

  1. Plug in manager
    • in terminal
      • mkdir ~p ~/.vim/bundle
      • cd ~/.vim/bundle
      • git clone https://github.com/<VundleVim>/Vundle.vim.git Vundle.vim
        • remove <> when typing command
    • in vimrc
      • under: boilerplate for plug in manager vundle
    • enabling vundle
      • vim ~/.vimrc
      • in .vimrc --> :PluginInstall
      • should open another window and show "done" when done, press "q" to exit
    • how to use:
      • everytime we want to install a new plugin for vim that's from github, specify:
        • format: Plugin '<gitub_account>/<repository_name>' between <call vundle#begin(...)> and <call vundle#end()> in vimrc
      • remove a plugin:
        • comment out the <Plugin ...> line in vimrc
        • run PluginClean in Vim (any instance)
      • repos will be downloaded into ~/.vim/plugged/
  2. formatting
    • in vimrc:
      • set incsearch -->> enable incremental search (automatically update matches in source file)
      • set hlsearch --> enable highlight search
      • can run :noh in vim instance to hide highlighted test
